## Your AI workforce, designed in the browser.

DeepSkills turns plain English into production-ready agents. No SDK juggling. No model lock-in. No 200-line config files.

> *"Show me top customers by region this quarter."*
>
> Your agent reads the question, queries your database, and renders the chart. Done.

---

## ✨ Why teams pick DeepSkills

🧠 **Design agents visually** — describe the role and the persona writes itself. Drag-and-drop skills onto a pipeline. No code required to ship.

⚡ **Bring your own AI** — OpenAI, Anthropic, Google, Groq, or Mistral. Switch providers with a click. Your keys, encrypted at rest, never leave the server.

🔌 **Talk to your real data** — connect Postgres, MySQL, files, or folders. Agents query live data and answer in context.

📊 **Answers as charts, instantly** — 18 chart types and a Tableau-style drag-and-drop builder. Or just describe a chart and watch the AI build it for you.

📦 **Export, don't escape** — every agent compiles to clean **TypeScript** *and* **Python** following the official [LangChain Deep Agents](https://docs.langchain.com) pattern. Run it anywhere — Vercel, AWS, your laptop.

🔒 **Secure by default** — two-step OTP login, AES-256-GCM encrypted API keys, idle-session timeout, and bcrypt-hashed passwords out of the box.

## 🚀 Get started in 60 seconds

```bash
git clone https://github.com/ravisekhar-design/DeepSkills.git
cd DeepSkills && npm install && npm run dev
```

Drop a free [Neon](https://neon.tech) Postgres URL into `.env`, add an AI provider key in **Settings → API Keys**, and you're live on `localhost:3000`.

Or skip setup entirely → **[Try it live](https://deep-skills.vercel.app)**.
